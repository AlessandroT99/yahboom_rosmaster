# yahboom_rosmaster_description

`yahboom_rosmaster_description` is a ROS 2 package intended to hold robot description assets for the Yahboom ROSMaster series.

## Purpose

This package is a placeholder for:

- URDF/XACRO robot descriptions
- mesh files
- launch files for visualization
- robot documentation and assets

## Build

Build this package from the workspace root:

```bash
cd ~/ros2_ws
colcon build --packages-select yahboom_rosmaster_description
```
